# Go To Station Widget

This project contains the source code for the "Go To Station" ArcGIS Experience Builder widget.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Linting**:
    ```bash
    npm run lint
    ```

3.  **Run Tests**:
    ```bash
    npm test
    ```

## Project Structure

-   `WidgetTemplate/src`: Source code for the widget.
-   `.github/workflows`: CI/CD configuration.
-   `docs/`: Additional documentation and verification guides.

## Development

This project is configured with ESLint and Prettier. Please ensure all code passes linting before committing.
